# SABDA 3D Asset Sourcing Guide

## What SABDA Needs
- 2K+ textures (hero objects), proper UV mapping, PBR materials
- Animation data (morph targets or skeletal) for moving objects
- Commercial licence covering public projection use
- GLB format preferred (direct Three.js), FBX needs conversion

## Tier 1 — Direct GLB / Easy Conversion

**Sketchfab** — https://sketchfab.com/store
Free + paid ($5-200). Preview in 3D before buying. GLB direct download. Largest animated creature selection.

**Poly Haven** — https://polyhaven.com
100% free CC0. Film-quality environments, trees, rocks, terrain, HDRIs. No creatures.

**Fab (Epic)** — https://fab.com
Replaced old UE Marketplace. Stunning quality. Some GLB, some need Blender conversion.

## Tier 2 — Great Quality, May Need Conversion

**TurboSquid** — https://www.turbosquid.com
$50-500+. CheckMate Pro = highest quality standard. Best for hero assets. FBX→GLB conversion needed.

**CGTrader** — https://www.cgtrader.com
Strong on architectural + nature. Good mid-range pricing. Mixed formats.

**RenderHub** — https://www.renderhub.com
Good animated animal selection. Nyilonelycompany birds listed here too.

## Tier 3 — Niche / Conversion Required

**Unity Asset Store** — https://assetstore.unity.com
Useful for Unity→NDI pipeline (flocking AI, particle packs). Not for Three.js direct.

**Blender Market / Superhive** — https://www.blendermarket.com
Great if .blend format, direct GLB export. Flock generators, procedural tools.

**ArtStation Marketplace** — https://www.artstation.com/marketplace
Multi-species packs (10+ birds in one). FBX/Max format, needs conversion.

**3DOcean** — https://3docean.net
Budget options, mixed quality. Point-cache animations.

- Premium: TurboSquid "Pelican Flying" $129, CheckMate Pro certified

**Flocks:** Nestaeric "Flock of Great Tits" ~$30, 86.9K tris, pre-animated flock. Unluck Software Bird Flock Bundle (Unity).

## Artist Directory
- **Nyilonelycompany** — GLB direct, 4K PBR, game-ready, $20-30 range. Best value.
- **Nestaeric** — Photorealistic master. $30-70. Best quality.
- **Holylights** — Multiple LODs, 4K PBR, $40 range.

## Conversion: FBX → GLB
Use Blender: Import FBX → verify materials → Export as GLB (Draco compression optional).
